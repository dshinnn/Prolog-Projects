Crossword.pl

This knowledge base plays a simple instance of a crossword puzzle and returns the position and orientation of the words in each of the empty crossword slots.

FamilyRules.pl

This code is a set of rules that uses the power of Prolog backtracking to identifies the relation of mothers, fathers, siblings, brothers, sisters, uncles, aunts, grandfathers, grandmothers, ancestors and descendants of a given knowledge base.  To test this code, you would need a family tree with the factlist composed of parent/2 facts.

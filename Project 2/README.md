Travel.pl

Given a factlist of byCar/3, byTrain/3 and byPlane/3 this Prolog program returns the paths allowed (if available) from one city to another and the total distance.

Arithmetic.pl

This Prolog program is an exercise to define our own predecessor/2, successor/2, addition/3, subtraction/3, multiplication/3 factorial/2, equal/2, greater/2, and less/2 rules.
